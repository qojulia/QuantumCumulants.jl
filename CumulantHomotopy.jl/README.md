# CumulantHomotopy.jl

CumulantHomotopy.jl is a research project for computing and validating stationary states of truncated quantum cumulant and moment hierarchies generated by QuantumCumulants.jl.

The central problem is not merely to find a root of a nonlinear stationary system. It is to understand how a finite hierarchy can recover the unique physical stationary state of the original infinite Liouvillian problem, even when each finite cumulant closure has many algebraic roots.

## Problem

The exact hierarchy of moments can be written schematically as an infinite linear system

```math
\dot y = A y,
```

where `y(O) = \langle O \rangle` is a normalized positive linear functional on the operator algebra. A unique stationary density operator means

```math
\ker A \cap \{\text{normalized positive functionals}\}
= \{y_{\mathrm{ss}}\}.
```

A cumulant truncation replaces the unresolved moments by a nonlinear reconstruction map. At order `n`,

```math
\dot x = f_n(x) = P_n A S_n(x),
```

and the stationary equations are

```math
F_n(x) = f_n(x) = 0.
```

The nonlinear closure manifold `S_n(x)` can create additional real and complex roots even when the exact Liouvillian has one stationary state. These roots may be unphysical, unstable, metastable, path dependent, or simply inconsistent with the next hierarchy order.

## Goal

CumulantHomotopy.jl aims to construct a sequence of physically selected finite order stationary approximations and determine whether their low order observables collapse to a unique limit.

The intended convergence statement is not

```math
\#\{x:F_n(x)=0\} \longrightarrow 1.
```

Instead, for a fixed low order projection `\pi_k`, we seek

```math
\operatorname{diam}\!\left(\pi_k \mathcal P_n\right)
\longrightarrow 0,
```

where `\mathcal P_n` is the set of stationary candidates that survive increasingly strong physical and cross order consistency tests.

Formal algebraic roots may remain distinct. What should collapse is the set of physically admissible predictions for fixed observables.

## Integrated method

The project combines three layers.

### 1. Candidate generation

Generate one or more stationary candidates using methods such as:

1. Newton or nonlinear least squares from physically informed initial states.
2. Long time integration of the truncated equations.
3. Continuation in physical parameters.
4. Continuation between hierarchy orders using the actual cumulant closure map.
5. Optional polynomial homotopy continuation for low order enumeration and reference tracking.
6. Pseudoarclength continuation near folds and closure induced bifurcations.

HomotopyContinuation.jl is one useful backend. It is not the definition of the method and is not required to be the first implementation dependency.

### 2. Physical validation and hierarchy collapse

Evaluate every candidate using information inherited from the original open quantum system:

1. Stationary residuals.
2. Adjoint and symmetry consistency.
3. Held out equations `\langle \mathcal L^\dagger O\rangle = 0` not used in the closure.
4. Moment matrix positivity and extendibility tests.
5. Energy, occupation, and other growth bounds.
6. Consistency with the next hierarchy order.
7. Convergence of shared low order moments.
8. Path dependence and branch clustering.

In parallel, construct a convex stationary moment envelope

```math
K_{n,q}
=
\left\{
 y:
 y(1)=1,
 \; y(\mathcal L^\dagger O)=0,
 \; M_q(y)\succeq0,
 \; \text{operator constraints hold}
\right\}.
```

This envelope does not replace cumulant roots. It provides an external physical reference against which roots can be tested, projected, and bounded.

### 3. Structure preserving closure research

Use the diagnostics above to improve the closure itself. Candidate approaches include:

1. Maximum entropy or exponential family reconstructions.
2. Invariant manifold and invariance defect corrections.
3. Thermodynamic projection for detailed balance systems.
4. Minimal representability corrections inspired by BBGKY purification.
5. Adaptive operator bases selected by the defect of invariance under `\mathcal L^\dagger`.
6. Auxiliary memory variables motivated by Mori-Zwanzig reduction.

This layer addresses the deeper question: how to build a finite phase space whose stationary structure better reflects the original Liouvillian.

## Continuation between hierarchy orders

Suppose `x` contains moments retained at order `n`, while `z` first appears at order `n+1`. The lower order closure predicts

```math
z = \Phi_n(x).
```

The correct lift is therefore not generally `z = 0`. Introduce closure deviation coordinates

```math
u = z - \Phi_n(x).
```

The lower order solution starts at `u = 0`, and an order homotopy or corrector gradually activates the complete order `n+1` stationary equations.

A useful next order defect is

```math
\delta_n
=
\left\|
W_{n+1}F_{n+1}\!\left(x_n,\Phi_n(x_n)\right)
\right\|.
```

This is an out of sample test. Every algebraic root has a small residual in its own closed system, but a spurious root may badly violate the next order equations.

## Coordinate representations

Realification is optional.

A complex polynomial backend may treat moments associated with `O` and `O^\dagger` as independent complex variables and check

```math
m_{O^\dagger} \approx m_O^*
```

at the endpoint.

A native physical solver may instead use reduced real coordinates that enforce adjoint relations exactly. The package should support both representations behind a common interface.

## Proposed workflow

```julia
using QuantumCumulants
using CumulantHomotopy

hierarchies = cumulant_hierarchies(problem; orders = 2:8)

result = stationary_sequence(
    hierarchies;
    candidates = CandidateMethods(
        NewtonCandidates(),
        ParameterContinuation(),
        OrderContinuation(),
    ),
    validation = PhysicalValidation(
        heldout_equations = true,
        moment_matrices = :adaptive,
        stationary_envelope = :optional,
    ),
    collapse = CollapseCriteria(
        observable_order = 2,
        compare_parity_subsequences = true,
    ),
)
```

A result should expose both candidates and evidence:

```julia
result.orders
result.candidates
result.selected
result.stationary_residuals
result.order_lift_defects
result.shared_moment_errors
result.moment_matrix_margins
result.distance_to_stationary_envelope
result.observable_intervals
result.branch_clusters
result.path_dependence
result.collapse_status
```

## Finite order unique estimates

The convex stationary envelope can provide one reproducible estimate at each order through a strictly convex selector, for example

```math
y_n^\star
=
\arg\min_{y\in K_{n,q}}
\frac12\|W(y-y_{\mathrm{ref}})\|^2.
```

When the feasible set is nonempty and closed, this selector is unique. The reference may be a cumulant candidate, the previous order result, or a maximum entropy estimate.

This does not prove that the selected pseudomoment vector is the exact state. It provides a unique finite order representative while the widths of rigorous or numerical observable intervals measure remaining ambiguity.

## Research foundations

The project draws on several established areas:

1. Moment closure and variational closure for stochastic systems.
2. Stationary finite state projection for infinite Markov chains.
3. Truncated moment problems and semidefinite moment hierarchies.
4. Stationary bootstrap methods for bosonic Lindblad systems.
5. BBGKY representability and purification corrections.
6. Invariant manifold and thermodynamic projection methods.
7. Mori-Zwanzig projection and memory effects.
8. Koopman and Carleman finite section approximations.
9. Numerical continuation and polynomial homotopy methods.

The detailed literature map and mathematical specification are in [docs/SPEC.md](docs/SPEC.md).

## Scope

The first objective is not a universal convergence theorem or complete high order root enumeration. It is a reproducible framework that:

1. Generates physically informed stationary candidates.
2. Tracks candidates across parameters and hierarchy orders.
3. Measures whether physical predictions collapse with increasing order.
4. Rejects or flags roots that violate realizability and held out constraints.
5. Provides a path toward structure preserving closures.
